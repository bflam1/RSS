Mend.io Unified Release Notes

Mend AppSec Platform: Version 26.2.2 (08-March-2026)

2026-03-08T00:00:00+00:00

New Features and Updates

  • Added a new EPSS Score column to the Dependencies Findings table, allowing users to view exploit likelihood at a glance and sort by risk. This enhancement streamlines vulnerability triage and prioritization by making EPSS data directly accessible without drilling into details.

  • Added a new pre-defined "Auditor" role that provides read-only access for service users requiring data visibility.

    • Auditor role assignments in Legacy SCA will automatically be reflected as such in the Mend AppSec Platform and vice versa.

  • Added support for filtering scans, reports, findings, and violations by the AI scan engine across the Mend AppSec Platform. The AI option is always visible in Scan Engine filters and enabled for customers with relevant entitlements.

  • UI Improvements:

    • Updated the table selection experience by moving the select all/deselect all controls into a single checkbox in the table header.

    • Redesigned and relocated the Refresh button for a more intuitive user experience.

    • Updated the Workflows and Reports pages by replacing filters with new filter buttons, making filtering more streamlined and user-friendly.

    • The Reports page now displays the "Delete" bulk action button only when a table row is selected, and the button text has been updated from Remove to Delete for improved clarity.

Resolved Issues

  • Fixed an issue where suppression comments longer than 255 characters were being truncated when retrieved via the UI, API, or CSV export, despite being saved correctly. Now, comments up to 500 characters are fully supported and displayed consistently across all interfaces.

Mend AI: Version 25.12.1 (05-January-2026)

2026-01-05T00:00:00+00:00

New Features and Updates

  • (Premium / Core) Introducing System Prompt Risk (open beta), a new detection and remediation offering by Mend AI, for mitigating risks posed by system prompts used in conversational AI interfaces.

    • A new System Prompt Risk table inventories system prompts and provides quick export/sharing and deep-link side panel for prompt context.

    • System Prompt Risks are integrated into the existing AI Security Risk Factors across Projects and Applications. The classification appears as a Conversational Interface chip, is filterable, and automatically participates in dashboards and workflows.

    • Remediation in the form of a hardened system prompt is available in the System Prompt side panel, providing clear, copy-ready remediation guidance for AppSec engineers and developers.

    • The AI Security Dashboard has been enriched with system prompt risk data.

    • System prompt risk data is also available via API.

Mend Unified Agent: Version 26.2.1 (23-February-2026)

2026-02-23T00:00:00+00:00

Resolved Issues

  • Fixed an issue where scanning large Docker container images in OCI tar format could fail silently due to unnecessary duplication of the .tar file, which exhausted disk space and resulted in incomplete scans with no reported dependencies.

  • Fixed an issue where Docker image scans failed when using the new Docker CLI output format. The Unified Agent now correctly parses image IDs, ensuring successful vulnerability scans for Docker images regardless of Docker CLI changes.
