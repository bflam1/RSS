Mend.io Unified Release Notes

Mend AppSec Platform: Version 25.7.2 (10-August-2025)

2025-08-10T00:00:00+00:00

New Features and Updates

  • Mend scans now include a "Reference" column, displaying the scanned branch (for Dependencies/Code) or image tag (for Containers).
    This enhancement improves visibility and context, and allows filtering scan results by branch or image tag.

  • Email notifications now default to opt-in for listed platform users, who can opt out via "My Profile".

Mend SCA: Version 25.7.2 (10-August-2025)

2025-08-10T00:00:00+00:00

New Features and Updates

  • The Dependencies SBOM report, previously available at the application/project level only, can now be generated at the organization level.

  • Improved license alignment with the SPDX format. This improvement includes the following changes, where in each of the following bullets, all instances of the former license in the platform have been replaced by the latter:

    • Eclipse 1.0 BSD 3

    • Jaxen Werken BSD 3

    • Cryptix General License BSD 2

    • Cup Parser Generator License Standard ML of New Jersey

Mend SAST: Version 25.7.2 (11-August-2025)

2025-08-11T00:00:00+00:00

New Features and Updates

  • The tag is now supported as taint source for Angular and React.

  • JavaScript/TypeScript analysis duration is now substantially reduced.

Resolved Issues

  • Resolved an issue in JavaScript/TypeScript and Python analysis where uncompilable code erroneously led to timeouts of the corresponding files.

  • Improved the progress bar display in the CLI during code scanning.

  • Fixed an issue where Cursor would occasionally lose the connection to the Mend MCP server.

Mend Container: Version 25.7.2 (10-August-2025)

2025-08-10T00:00:00+00:00

New Features and Updates

  • Automation Workflow updates for improved control and effective handling of container findings:

    • A Layer Group condition for the Container Analysis triggering event in Automation Workflows is now available, including values for Base Layers and Application Layers.

    • A Library Type condition for the Container Analysis triggering event in Automation Workflows is now available, including values for the detected librarys programming language or package manager (e.g., Debian, Maven).

Mend Unified Agent: Version 25.7.2 (11-August-2025)

2025-08-11T00:00:00+00:00

Resolved Issues

  • Improved the user experience by updating the screening process so only files that are used for dependency resolution will be logged, preventing misleading messages.

  • Fixed an issue where system memory was unnecessarily seized by the Gradle daemon after executing a Gradle script. Memory handling has been improved for increased efficiency.

Mend Developer Platform: Version 25.7.2 (10-August-2025)

2025-08-10T00:00:00+00:00

New Features and Updates

  • UI improvements:

    • Added a new column to the repositories table, which shows when the last job (scan) was executed for each repository.

    • Removed the Installation date and Availability columns from the repositories table.

    • Errors and warning indicators are now displayed in the second column instead of the first.

  • The Azure DevOps integration now enables Azure DevOps organization administrators to uninstall any project in their organization from the Developer Platform.

  • The Bitbucket integration now enables users to configure multiple workspace tokens which will be used in rotation, to avoid hitting Bitbuckets API rate limits.
