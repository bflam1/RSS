Mend.io Unified Release Notes

Mend AppSec Platform: Version 25.8.2 (07-September-2025)

2025-09-07T00:00:00+00:00

New Features and Updates

  • Added the ability to define custom SLA values for violations in automation workflows, ranging from 1 - 1000 days.

  • The Mend AI Native AppSec platform now supports IP allowlisting, configurable via the Administration > General tab. Administrators can configure access by specifying exact IP addresses or CIDR ranges, blocking unauthorized IPs from accessing the API or the platform UI.

Mend SCA: Version 25.8.2 (07-September-2025)

2025-09-07T00:00:00+00:00

New Features and Updates

  • Introducing three new SCA conditions for enhanced workflows:

    • Trigger actions for direct dependencies.

    • Detect components with multiple licenses to manage compliance risks.

    • Prioritize vulnerabilities with publicly available exploits for remediation.

  • Users can now group findings in the Findings tab within the Direct Library side-panel. This feature enables better analysis of violations by organizing data into collapsible/expandable Library Rows, displaying aggregated information and individual CVE details efficiently.

Mend SAST: Version 25.8.2-HF1 (14-September-2025)

2025-09-14T00:00:00+00:00

Resolved Issues

  • Addressed an issue where execution of C/C++ analysis failed on MacOS with Intel x86 architecture because of a missing library.

  • Improved the performance of PHP analysis, especially when processing large PHP files.

Mend AI: Version 25.8.2 (07-September-2025)

2025-09-07T00:00:00+00:00

New Features and Updates

  • (Premium) Introducing a centralized dashboard for security managers to visualize and manage AI security risks. Gain single-view risk assessments, prioritize vulnerability remediation, and track security metrics. Components include AI and project widgets, with enhancements for seamless navigation and detailed risk insights.

  • (Premium) Academic security findings are now integrated into the AI Models table, offering users immediate visibility of model-specific vulnerabilities. This enhancement supports proactive risk assessment, seamless workflow integration, and provides detailed contextual insights, while maintaining existing navigation patterns and preparing for future integration of additional security intelligence sources.

  • (Premium) Added an AI Security Findings column to Projects and Applications tables showing severity distribution (Critical/High/Medium/Low). One-click drill-down opens the AI Models view for the selected item to accelerate investigation.

  • (Premium) Introducing a Severity Distribution widget that summarizes AI security findings by severity. Clicking a severity opens the Projects view pre-filtered and sorted to highlight the most impacted projects for fast triage.

  • Added a side panel in the AI Technologies table that shows the full list of related AI libraries and details without truncation, enabling faster review and decision making.
