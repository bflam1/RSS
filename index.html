Mend SCA: Version 25.11.1 (30-November-2025)

2025-11-30T00:00:00+00:00

New Features and Updates

  • SBOM improvements:

    • Enhanced the SBOM import experience by introducing a clear, step-by-step process with status indicators, user-friendly messages, and actionable guidance for resolving validation or structure issues. Added a 5 MB log view limit with download options for larger logs, and ensured consistent, readable UX for both SPDX and CycloneDX formats.

    • Added a new field to the license side panel that displays the SPDX identifier for each license, making it easier for users to identify licenses by their standard SPDX-ID.

    • Added export to CSV and refresh options to SBOM import logs. These enhancements make it easier to analyze, filter, and share import results, improving troubleshooting and overall usability.

  • Streamline library management by consolidating duplicate entries of direct libraries into a single row on the Direct Libraries page. Display the riskiest instance's data and allow pathway navigation via a new dropdown. Ensure actions affect all paths with clear user confirmations.

Mend SAST: Version 25.11.1 (01-December-2025)

2025-12-01T00:00:00+00:00

New Features and Updates

  • The Agentic SAST detection now reports all detected findings in the generated code at once instead of just one finding per invocation.

  • Added low probability taint sources for file, database, console, and environment variable reads in Rust, improving analysis accuracy.

  • Reclassified CWE-242: Use of Inherently Dangerous Function and CWE-676: Use of Potentially Dangerous Function to low probability category for C/C++, enhancing scan accuracy when scanning with the default configuration.

  • Further improved detection accuracy for C/C++ gen 2, most notably for the following CWEs:

    • CWE-121: Buffer Overflow

    • CWE-369: Divide by Zero

    • CWE-606: Loop Denial of Service

    • CWE-611: XML External Entity (XXE) Injection

    • CWE-789: Uncontrolled Memory Allocation

    Additionally, reading from local files has been added as a low probability taint source.

  • Organizations are now sorted alphabetically in the Mend CLIs organization selection step, making it easier and faster to navigate large lists.

Resolved Issues

  • Fixed an issue in the Classic Repository Integrations where SAST scans on pull requests targeting an empty main branch produced misleading results and incorrectly reported a successful scan status.

Mend Container: Version 25.10.3 (16-November-2025)

2025-11-16T00:00:00+00:00

New Features and Updates

  • General CLI container scan performance improvements.

Resolved Issues

  • Resolved a a log output issue during scans: By removing specific code lines, log messages no longer print to STDOUT/STDERR when MEND_LOG_LEVEL is set. Console logging remains at ERROR level, except in development mode.

  • Resolved an issue in Desktop Printing Software for Linux where HTML reports for Container Image Attribution displayed an incorrect number of libraries, exceeding the project's parameters when generating reports in the customer's environment.

  • Fixed an issue where violations generated by workflows were not automatically resolved when their associated findings were suppressed. Now, violations are promptly resolved across all scan types when findings are suppressed, ensuring consistent and immediate updates.

  • Fixed an issue where users with the Security Analyst role were unable to suppress container findings via API v3 without admin permissions. Suppression can now be performed by users with appropriate non-admin roles.
