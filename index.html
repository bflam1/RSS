Mend.io Unified Release Notes

Mend AppSec Platform: Version 25.11.2 (14-December-2025)

2025-12-14T00:00:00+00:00

New Features and Updates

  • (SCA) Added support for SBOM import through API 3.0, enabling users to upload SBOM files within the application context. A dedicated logs API provides enhanced traceability and integration.

  • (SCA) Added a new Reports tab in the Accounts view, enabling account administrators to generate and manage Inventory, Security Alerts by Library, and Security Alerts by Vulnerability reports at the account level. Reports use an asynchronous flow with updated email notifications and permissions aligned to the Legacy SCA model.

  • (SCA) Legal and Compliance workflows now support Jira Issue creation, aligning with other workflows in the Mend AppSec Platform.

    • The Create Issue action for legal findings is available in the OS Inventory table as well as Automation Workflows with Licenses conditions.

Mend SAST: Version 25.11.1 (01-December-2025)

2025-12-01T00:00:00+00:00

New Features and Updates

  • The Agentic SAST detection now reports all detected findings in the generated code at once instead of just one finding per invocation.

  • Added low probability taint sources for file, database, console, and environment variable reads in Rust, improving analysis accuracy.

  • Reclassified CWE-242: Use of Inherently Dangerous Function and CWE-676: Use of Potentially Dangerous Function to low probability category for C/C++, enhancing scan accuracy when scanning with the default configuration.

  • Further improved detection accuracy for C/C++ gen 2, most notably for the following CWEs:

    • CWE-121: Buffer Overflow

    • CWE-369: Divide by Zero

    • CWE-606: Loop Denial of Service

    • CWE-611: XML External Entity (XXE) Injection

    • CWE-789: Uncontrolled Memory Allocation

    Additionally, reading from local files has been added as a low probability taint source.

  • Organizations are now sorted alphabetically in the Mend CLIs organization selection step, making it easier and faster to navigate large lists.

Resolved Issues

  • Fixed an issue in the Classic Repository Integrations where SAST scans on pull requests targeting an empty main branch produced misleading results and incorrectly reported a successful scan status.

Mend Unified Agent: Version 25.11.2 (15-December-2025)

2025-12-15T00:00:00+00:00

Resolved Issues

  • Fixed an issue where invalid system paths in certain PNPM and Yarn projects caused Reachability analysis to fail.

  • Fixed an issue where Go projects with build errors caused all dependencies to appear as direct in the UI, resulting in the loss of transitive dependency hierarchy and breaking vulnerability path analysis. Dependency relationships are now accurately reported even when fallback mechanisms are triggered.

  • Fixed an issue where Erlang dependencies could not be resolved during project scans, resulting in incomplete dependency trees.

  • Fixed an issue where Elixir dependencies could not be resolved during project scans, under certain conditions.
